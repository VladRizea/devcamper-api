# Devcamper api

> Backend api for the Devcamper application. Wich is a bootcamp directory website

## Usage

Rename "config.env.env" to "config.env" and update the values to your own

## Install dependencies

```bash
npm install
```

## Run app

```bash
# Run in dev mode
npm run dev

# Run in production mode
npm run start
```
